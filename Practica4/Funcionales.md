# Requerimientos Funcionales - Fase 2 Delivereats

## Gestión de Usuarios y Autenticación
### RF-01: El sistema debe permitir el registro de usuarios con rol CLIENTE, RESTAURANTE, REPARTIDOR y ADMINISTRADOR.
### RF-02: El sistema debe permitir el inicio de sesión mediante email y contraseña.
### RF-03: El sistema debe permitir la actualización de perfiles de usuario.
### RF-04: El sistema debe soportar recuperación de contraseña mediante correo.

## Gestión de Restaurantes
### RF-05: El administrador podrá crear, editar y eliminar restaurantes.
### RF-06: El administrador podrá asignar horarios de operación a restaurantes.
### RF-07: Los restaurantes podrán gestionar (CRUD) sus ítems de menú.
### RF-08: Los restaurantes podrán actualizar disponibilidad de productos en tiempo real.

## Experiencia del Cliente
### RF-09: El cliente podrá visualizar restaurantes y menús disponibles.
### RF-10: El cliente podrá filtrar restaurantes por categoría, calificación y distancia.
### RF-11: El cliente podrá crear una orden con uno o más productos.
### RF-12: El cliente podrá ver el historial de órdenes previas.

## Gestión de Órdenes
### RF-13: El restaurante podrá aceptar o rechazar una orden.
### RF-14: El restaurante podrá estimar el tiempo de preparación de una orden.
### RF-15: El repartidor podrá aceptar una orden para entrega.
### RF-16: El repartidor podrá actualizar el estado de la entrega en tiempo real.

## Notificaciones y Comunicación
### RF-17: El cliente recibirá notificaciones por correo según el estado de la orden.
### RF-18: El cliente recibirá notificaciones en tiempo real sobre cambios en el estado de su orden.
### RF-19: El restaurante recibirá notificaciones de nuevas órdenes entrantes.
### RF-20: El repartidor recibirá notificaciones de órdenes disponibles para entrega.

## Calificaciones y Retroalimentación
### RF-21: El cliente podrá calificar restaurantes, repartidores y órdenes.
### RF-22: El cliente podrá dejar comentarios en sus calificaciones.
### RF-23: El sistema debe mostrar promedios de calificación en perfiles de restaurantes y repartidores.

## Reportes y Análisis
### RF-24: El administrador podrá generar reportes de ventas por restaurante.
### RF-25: El administrador podrá visualizar métricas de desempeño del sistema.
